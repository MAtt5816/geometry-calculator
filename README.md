# GeometryCalculator

A simple calculator of basic parameters of geometric figures and solids.

## Implemented functions:
### Solids:
1. Ellipsoid (**area & volume**)
    * general ellipsoid
    * rotational ellipsoid (spheroid)
    * sphere
